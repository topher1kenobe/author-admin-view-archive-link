Creates a View Archive link in the list of authors on the Users admin page.

== Description ==
In the admin area, when viewing the Users list, each user has an Edit link.  This plugin adds a View Archive link that goes to the front of the site to that user\'s archive page.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the `author-admin-view-archive-link` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
1. Create new or Edit a Post.
